Internet et le web

Internet et le WWW

Internet est un ensemble de réseaux informatiques interconnectés à travers le monde, qui est accessible au grand public.

Parmi les réseaux interconnectés il y a ceux d'entreprises, de gouvernements, d'universités, de maisons, etc.

Ces réseaux son reliés au niveau mondial par un ensemble de câble sous-marins qui couvrent quasiment tout le globe (Voir Figure suivante).

Sur Internet il y a deux catégories d'équipements : les équipements terminaux et les équipements intermédiaires.

Les équipements terminaux sont au bout des communications et les équipements intermédiaires s'occupent de l'acheminement des données à travers le réseau.

Une classe d'équipements terminaux fournit les services (Web, Partage de Fichier, Base de données, Mailing) : les serveurs.

La seconde classe d'équipements terminaux sont directement utilisés par les internautes pour accéder aux services du réseau Internet: Ordinateurs, Smartphone, Tablettes, etc.

Les équipements intermédiaires sont les commutateurs, les routeurs, les modems, ... Ces équipements sont gérés en général pas les Fournisseurs d'Accès Internet (FAI) ou opérateur de télécommunication.

Pour vous connecter à Internet à partir de chez vous, vous êtes raccordé au réseau de votre fournisseur d'accès Internet grâce à un modem qui est lui raccordé au grand réseau International grâce aux câbles sous-marins.

Le World Wide Web ou W3 (ou le web tout simplement) est le service d'Internet qui permet d'accéder à l'ensemble des sites Web publiques du monde.

Contrairement à l'usage commun, le Web est différent de l'Internet : le premier est un service qui permet d'accéder aux sites et le second est l'infrastructure qui permet de déployer un tel service.

Le Web est en fait un protocole d'échanges de documents (pages web) entre les équipements terminaux d'internet.

Sur le plan architectural, le Web est composée de quatre grands blocs fonctionnels :

  • Le modèle de communication client-serveur défini le rôle des nœuds (équipements) dans le service en deux catégories : les serveurs et les clients.

  • Le langage de structuration du contenu HTML accompagné d'un langage de mise en forme (CSS)

  • Les URLs pour l'adressage (identification) des ressources du Web

  • Le protocole HTTP pour la transmission des données

Un navigateur Web, également appelé navigateur tout court, est un programme qui interprète et affiche les pages Web. IL permet de visualiser et d'interagir avec

une page Web.

Un Serveur Web désigne également le logiciel utilisé sur le serveur pour exécuter les requêtes HTTP.

Le Langage HTML est un langage informatique qui permet de créer et structurer des pages Web. Toutes les pages Web sont faites de code HTML.

Le Protocole HTTP (HyperText Transfer Protocol), littéralement « protocole de transfert hypertexte » — est un protocole de communication client-serveur développé pour le World Wide Web.

Une URL (Uniform Resource Locator), littéralement « localisateur uniforme de ressource »), auquel se substitue informellement le terme adresse web, désigne une chaîne de caractères au format: proto://machine:port/chemin/vers/la/ressource
