Mise en forme de pages HTML avec CSS3

CSS (Cascading Style Sheets), est aussi un langage de balises qui vient compléter le HTML.

Son rôle est de gérer la mise en forme des pages HTML:

  • choisir la couleur, la police, la taille, ... de votre texte.

  • définir les bordures, le fond, ... des pages

  • faire la mise en page du site. Par exemple, vous pourrez dire : je veux que mon menu soit à gauche et occupe telle largeur, que l'en-tête de mon site soit calé en haut et qu'il soit toujours visible, etc.

Grâce au HTML, on peut rédiger le contenu d'un site mais il est brut de décoffrage. Le CSS vient compléter ce code pour mettre en forme tout cela et donner au contenu l'apparence que l'on souhaite.

Il existe trois façons de mettre du CSS dans une page Web :

  • dans un fichier portant l'extension .css (méthode la plus recommandée) ;

  • dans l'en-tête <head> du fichier HTML ;

  • directement dans les balises du fichier HTML via un attribut style (méthode la moins recommandée).

La méthode la plus pratique pour écrire du code CSS est de la faire dans un fichier à part ayant l'extension .css (contrairement aux fichiers HTML qui ont 'extension .html). Cela permet d'éviter de tout mélanger dans un même fichier.

En utilisant cette méthode, il faut, dans la balise <link> (contenue dans la balise <head>), indiquer le ou les fichier(s) .css à utiliser. Exemple:  

<link rel="stylesheet" href="style.css" /> indique que le fichier HTML est associé à un fichier appelé style.css et chargé de la mise en forme de la page HTML.

Dans un code CSS, on trouve trois éléments différents :

  • Des noms de balises : on écrit les noms des balises dont on veut modifier l'apparence. Par exemple, si on veut modifier l'apparence de tous les paragraphes <p>, on doit écrire p.

  • Des propriétés CSS : les « effets de style » de la page sont rangés dans des propriétés. Il y a par exemple la propriété color qui permet d'indiquer la couleur du texte, font-size qui permet d'indiquer la taille du texte, etc.

  • Les valeurs : pour chaque propriété CSS, on doit indiquer une valeur. Par exemple, pour la propriété color, il faut indiquer le nom de la couleur. Pour font-size, il faut indiquer quelle taille on veut, etc.

Dans les exemples précédents, il est indiqué que  tous les titres h1 vont être coloriés  en bleus. Si on veut  faire en sorte que seuls certains titres (bien précis)  soient ciblés et écrits d'une manière différentes, il faut utiliser ces attributs spéciaux qui fonctionnent sur toutes les balises :
  • l'attribut class ;
  • l'attribut id.
Les attributs class et id fonctionne exactement de la même manière à un détail près : un id ne peut être utilisé qu'une fois dans le code HTML d'une page (deux balises HTML ne peuvent pas avoir la même valeur pour l'attribut id. Une manière d'identifier une balise de manière unique dans une page), alors qu'une class peut l'être plusieurs fois (plusieurs balises peuvent avoir la même valeur pour l'attribut class ; une manière de donner la même apparence à toutes les balises ayant la même valeur pour l'attribut class).
Dans l'exemple suivant, introduction est le nom donné à l'attribut class et permet d'identifier les balises qui le comportent où elles se trouvent :

<body>
2

3
   <p class="introduction">Bonjour et bienvenue sur mon site !</p>
4
   <p>Pour le moment, mon site est <em>vide</em>. Patientez encore un peu !</p>
5
</body>

Il arrivera parfois qu'on ait besoin d'appliquer une class (ou un id) à certains mots qui, à l'origine, ne sont pas entourés par des balises.
En effet, le problème de class, c'est qu'il s'agit d'un attribut. On ne peut donc en mettre que sur une balise. Si, par exemple, ont veut modifier uniquement le mot «bienvenue» dans le paragraphe suivant :
1
<p>Bonjour et bienvenue sur mon site !</p>
<p>Bonjour et bienvenue sur mon site !</p>
Il existe deux balises dites universelles (qui n'ont aucune signification particulière) pour traiter ce genre de situation:
  • <span> </span>
  • <div> </div>
Il y a une différence minime (mais significative !) entre ces deux balises :
  • <span> </span>. C'est une balise de type inline, c'est-à-dire une balise que l'on place au sein d'un paragraphe de texte, pour sélectionner certains mots uniquement. Les balises <strong> et <em> sont de la même famille. Cette balise s'utilise donc au milieu d'un paragraphe, comme dans l'exemple suivant:
1
<p>Bonjour et <span class="introduction">bienvenue<span> sur mon site !</p>
<p>Bonjour et <span class="introduction">bienvenue<span> sur mon site !</p>
  • <div> </div>. C'est une balise de type block, qui entoure un bloc de texte. Les balises <p>, <h1>, etc. sont de la même famille. Ces balises ont quelque chose en commun : elles créent un nouveau « bloc » dans la page et provoquent donc obligatoirement un retour à la ligne. <div> est une balise fréquemment utilisée dans la construction d'un design.
Les Sélecteurs:
En CSS, le plus difficile est de savoir cibler le texte dont on veut changer la forme. Pour cibler (on dit « sélectionner ») les éléments de la page à modifier, on utilise ce qu'on appelle des sélecteurs:
signifie qu'« on veux toucher tous les paragraphes ».
signifie qu'« on veut toucher tous les titres et tous les textes importants »: sélection de deux balises d'un coup.
Sélectionne toutes les balises sans exception. On l'appelle le sélecteur universel.
Sélectionne toutes les balises <em> situées à l'intérieur d'une balise <h3>. Notez qu'il n'y a pas de virgule entre les deux noms de balises. Exemple: <h3>Titre avec <em>texte important</em></h3>
Sélectionne la première balise <p> située après un titre <h3>. Exemple: <h3>Titre</h3> <p>Paragraphe</p>
